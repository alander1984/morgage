### Инструкция по развертыванию ###

Развертывание прикладного ПО зависит от особенностей используемой операционной системы. Для развертывания приложения должны быть установлены:

 1. ruby (>=v2.2.5)
 2. rails (>=v4.2.4)
 3. PostgreSQL (>=v9.4)

Для установки приложения необходимо выполнить следующие действия:

 1. Распаковать архив с исходниками, или выполнить команду
 

  `git clone https://github.com/alander1984/morgage.git`
 2. Перейти в каталог morgage и выполнить установку необходимых для проекта пакетов

  `bundle install`
 3. В зависимости от используемого сервера БД (локально или удаленно) - необходимо создать на нем схему `mortgage_development` и настроить подключение в файле проекта `config\database.yml`: указать имя пользователя, пароль (опционально адрес сервера БД)

 
 4. Выполнить миграции создания структуры БД

  `rake db:migate`

 5. Выполнить миграции первичного наполнения данных

  `rake db:seed`
 6. Запустить приложение стандартной командой

  `rails s -b 0.0.0.0` (остальные параметры запуска сервера опциональны)
